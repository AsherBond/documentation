Eucalyptus Dashboard Overview

Understanding the Eucalyptus Dashboard

This section explains the components of the Dashboard screen.

The Dashboard screen has the following areas:

Header
This area includes the logo, the link to a user profile setting menu, and the big search box.
Footer
This area shows status information, a log button to show a popup log window and the current version of Eucalyptus.
Quick Links
This area provides quick links to various contents of the dashboard. It is a central place for users to explore what they can do with the dashboard.
Main Content
This area shows the main content of certain type of information. It is usually the list of search results.
Properties
For the search result list content, e.g. the list of all accounts, click on one of the list item to show the detailed properties of that item. You may be able to change values or invoke actions for the properties. Click the “x” next to the “Properties” title to hide the area.
