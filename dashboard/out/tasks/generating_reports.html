Generating Usage Reports

This section shows you how to generate usage reports using the Eucalyptus Dashboard.

Context for the current task

  1. In the Dashboard, click Usage Report in the Quick Links section. The Usage Report page displays.
  2. In the Start date field, select the date to start
  3. Click the Generate button.
The usage report displays, reflecting your choices in the report.
