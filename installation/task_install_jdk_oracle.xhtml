Install the Oracle JDK

Procedure 1. To install the Oracle JDK

  1. Go to http://www.oracle.com/technetwork/java/javase/downloads/index.html.

  2. Select Java Platform (JDK).

    The Java SE Downloads page is updated with a list of packages to download.

  3. Accept the license agreement.

  4. Select either Linux x86 or Linux x64 and whichever installer you prefer.

    The package you chose is downloaded.

  5. Open a terminal window and type the following two commands:

    chmod 0700 jdk*.bin
./jdk*.bin

    The first command protects the downloaded package from access. The second command installs the files to a directory labeled "jdk" and the current version.

  6. Create the directory/opt/packages/jdk:

    mkdir -p /opt/packages/jdk
