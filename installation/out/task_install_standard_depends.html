Installing Eucalyptus Dependencies

Install Standard Dependencies

Install the Eucalyptus-prepared dependency RPMs on all machines, both front-end and node machines.

Note: Remove any preexisting copies of these dependencies before you install. The install process returns an error message if it finds multiple copies of a dependency.
In the terminal window, type the following commands: 
[root@clc]# cd eucalyptus-deps*
[root@clc]# yum install -y *.rpm --nogpgcheck
[root@node1]# cd eucalyptus-deps*
[root@node1]# yum install -y *.rpm --nogpgcheck
